# Landing calendar: Community messages filter

- Added a clearly labelled **Community messages** checkbox to the landing-page calendar toolbar.
- The filter is on by default and can be unticked to remove Communities messages from the calendar.
- The choice is saved in the existing browser configuration and restored after refresh.
- Removed the ambiguous duplicate **Comms** layer chip so there is one clear control for this behaviour.
- The Communities tab and its messages are not changed or deleted; this only controls visibility on the landing calendar.
