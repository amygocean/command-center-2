# Task completion celebrations

- Every genuine task completion now triggers a celebratory confetti burst and a short completion card after Asana confirms the save.
- Communities messages and shoot days use context-specific wording.
- Campaign subtasks use a smaller, quicker celebration.
- Completing the final shot in a run sheet produces one larger “That’s a wrap” moment rather than overlapping effects.
- Failed Asana writes roll the task back and never celebrate a completion that did not save.
- Reopening a task does not trigger confetti.
- People who prefer reduced motion see the completion card without animated confetti.
- Completion celebrations can be switched off under Settings → Experience. The setting is personal to that browser.
