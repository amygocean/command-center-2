# Safe task board changes

- Opening or editing a task now keeps it on its current Asana board by default.
- The drawer shows the task's current board as read-only context.
- Moving a task requires ticking **Move this task to another board** and selecting a destination.
- The destination section selector only appears after a destination board is chosen.
- Tasks opened from The Girls now retain their current project GID when Asana provides one.
- Tasks that currently live only in My Tasks can be added to a board without trying to remove a non-existent current project.
